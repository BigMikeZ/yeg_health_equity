# Edmonton Health Equity: The Social Gradient of Diabetes
**Author:** Mike Zhang
**Status:** In Progress (Week 1: Data Acquisition & Cleaning)

## Project Goal
To analyze the relationship between neighborhood socioeconomic status (Income) and chronic health outcomes (Diabetes) in Edmonton, AB.

## Tech Stack
- **Language:** R 
- **Version Control:** Git/GitHub
- **Data Sources:** City of Edmonton Open Data, Alberta IHDA
